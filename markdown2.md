## Inserarea codului de progamare in Markdown

### Cod ` in linie `

**HTML :**

Iata un fragment de cod scris pe aceeasi linie : ` a = b**c`
<code> a=b**c </code>

### Cod scris pe mai multe linii

<pre>
import pandas as pd
import numpy as np

a = 2
b = 3
</pre>

## Inserarea sistemelor evidentiate (blockquote)

**Markdown**
 
> Acesta este un text evidentiat.

**HTML :**
<blockquote>
Acesta este un text evidentiat.
</blockquote>
